xsh

A lightweight utility CLI for Linux and macOS. Package management, port scanning, DNS, WHOIS, and traceroute in one tool.

Install
$sudo curl -fsSL https://raw.githubusercontent.com/srddeveloper/xsh/refs/heads/main/install.sh -o /usr/local/bin/xsh && sudo chmod +x /usr/local/bin/xsh
Commands
help Show the help menu
install <package> Install a package
update all Update all packages
setup all Install all basic system packages
scan <ip> Scan common ports with banners
port <ip> <port> Test a specific port with banner
dns <host> DNS lookup
whois <host> WHOIS lookup
trace <host> Traceroute

Requires sudo on some systems. Installs to /usr/local/bin.
